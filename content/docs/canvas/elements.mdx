# Canvas Elements

Growhouse supports various element types to help you create rich, visual content.

## Text Elements

Add rich text with full formatting support:

- **Headings** (H1, H2, H3)
- **Bold**, *italic*, ~~strikethrough~~
- Bullet and numbered lists
- Links and mentions

### Creating Text

1. Click the **Text** tool or press `T`
2. Click on the canvas
3. Start typing

## Image Elements

Upload or paste images directly onto the canvas.

### Supported Formats

- PNG, JPG, JPEG, GIF, WebP
- SVG (vector graphics)
- HEIC (auto-converted)

### Adding Images

- **Upload**: Drag and drop from your computer
- **Paste**: Copy an image and `Cmd/Ctrl + V`
- **URL**: Paste an image URL directly

## Video Elements

Embed videos from popular platforms:

- YouTube
- Vimeo
- Loom
- Direct video URLs

## Embed Elements

Add content from external sources:

- Figma designs
- Twitter/X posts
- Instagram posts
- TikTok videos
- And more...

## Annotations

Add visual annotations to highlight important areas:

- Arrows and lines
- Shapes (rectangles, circles)
- Sticky notes
- Comments
