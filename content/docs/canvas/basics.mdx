# Canvas Basics

The canvas is the core of Growhouse - a flexible workspace where you create and organize content.

## Navigation

### Pan & Zoom

- **Pan**: Click and drag on empty space, or hold `Space` + drag
- **Zoom In**: `Cmd/Ctrl + =` or scroll up
- **Zoom Out**: `Cmd/Ctrl + -` or scroll down
- **Fit to Screen**: `Cmd/Ctrl + 0`

### Selection

- **Select**: Click on an element
- **Multi-select**: Hold `Shift` and click multiple elements
- **Box Select**: Click and drag on empty space to create a selection box

## Working with Elements

### Adding Elements

1. Click the **"+"** button in the toolbar
2. Choose the element type
3. Click on the canvas to place it

### Moving Elements

- Drag elements to reposition them
- Hold `Shift` while dragging to constrain movement
- Use arrow keys for precise nudging

### Resizing Elements

- Drag corner handles to resize
- Hold `Shift` to maintain aspect ratio
- Hold `Alt/Option` to resize from center

## Keyboard Shortcuts

| Action | Shortcut |
|--------|----------|
| Undo | `Cmd/Ctrl + Z` |
| Redo | `Cmd/Ctrl + Shift + Z` |
| Copy | `Cmd/Ctrl + C` |
| Paste | `Cmd/Ctrl + V` |
| Delete | `Backspace` or `Delete` |
| Select All | `Cmd/Ctrl + A` |
